# 系统特点
简单、实用、归纳、管理。<br>
**不需要数据库依赖，秉承低门槛、易操作、0维护的特点，放入 md 文件夹即可一键生成。**

# 适用场景
文学爱好者、作家、博主、IT程序员、摄影爱好者、专业学科定制等。

# 特别注意
本博客系统不提供 `MarkDown编辑器` 功能。
