# 目录说明

/blog ... 存放博客网页文件
/data ... 存放系统生成的站点数据
/index ... 由系统生成的index列表
/md ... 用户 MarkDown 文件放置文件夹
/set ... 站点配置
/set/api ... 系统功能
/static ... 层叠样式表、JavaScript及第三方依赖
/tpl ... 模板目录
/favicon.ico ... 博客图标
/index.html ... 系统生成的博客主页
