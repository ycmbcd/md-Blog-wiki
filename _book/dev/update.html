# 系统更新

保留以下目录，更新其他目录即可。

/data ... 参数配置及站点数据文件夹
/md ... 您的 MarkDown 文件夹
/tpl ... （可选）如模板自定义，请保留
/static ... （可选）如模板自定义，请保留
